# Handoff files

Drop a file here whenever you pause mid-task or pass work to the other collaborator.

**Naming:** `YYYY-MM-DD-<short-topic>.md` (e.g. `2026-04-20-nextjs-port-kickoff.md`).

**Include:**
- **Goal** — what you were trying to accomplish
- **Done** — what's actually committed
- **Next** — the immediate next step for whoever picks it up
- **Gotchas** — anything surprising, broken, or half-wired
- **Files touched** — quick list so Claude can jump straight in

The next Claude session will read these automatically when briefed on a handoff.
